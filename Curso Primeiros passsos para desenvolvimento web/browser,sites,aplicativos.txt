browser - programa para visualizar sites/interpreta linguagens de programação/marcação e conteudo multimedia (diversos tipos de sites) - feitas em diversas linguagens de programação

Aplicativo - forma diferente de acessar um conteudo de um site

statico (fisíco) dinamico (software que estão no servidor fisico
tipos: arquivos, aplicações, banco de dados ou um que tem tudo junto
